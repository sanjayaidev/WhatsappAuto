Apps Script Setup
The Apps Script file reads your lead config from the Config tab automatically. Steps to connect:
01Open your Google Sheet → Extensions → Apps Script
02Paste the AutoSend.gs file contents, set GOWA_URL at the top
03Save config here on the dashboard → it writes to the Config tab
04In Apps Script menu → AutoSend → Setup Auto-Trigger (run once)
05New leads in Sheet1 now auto-receive WhatsApp messages 🎉
